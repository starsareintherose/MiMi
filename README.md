## Functions

Convert any formats listed in `Accepted formats` to each other.

## Arguments

- -h	--help;

- -i	--input		${filename};

- -o	--output	${filename};

## Accepted formats

- fas	fasta

- nex	nexus

- phy	phylip

- tnt	ss

## Compile

```
g++ main.cpp -o mimi
```
